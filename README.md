# 🌤️ Modern Weather App / Időjárás App (Python)

*[🇬🇧 Scroll down for the English version](#-english-version)*

---

## 🇭🇺 Magyar Verzió

Egy egyszerű, de látványos grafikus (GUI) időjárás-jelentő alkalmazás, amely a Python és az OpenWeatherMap API segítségével készült. A program a beírt város alapján lekéri a jelenlegi hőmérsékletet, az időjárás állapotát, és a kapott adatok alapján még öltözködési tippet is ad!

### 🚀 Funkciók
* Modern felület: A customtkinter könyvtárnak köszönhetően letisztult, sötét témájú dizájn.
* Valós idejű adatok: Élő időjárás lekérdezés a világ bármely pontjáról.
* Okos asszisztens: Személyre szabott üzenetek a kinti hőmérséklet függvényében.

### 🛠️ Telepítés és Használat

A program futtatásához a gépedre telepítve kell lennie a Pythonnak.

1. Könyvtárak telepítése
Nyiss egy parancssort, és futtasd az alábbi parancsot a szükséges modulok letöltéséhez:
pip install requests customtkinter

2. API Kulcs beállítása
* Regisztrálj egy ingyenes fiókot az OpenWeatherMap weboldalán.
* Generálj egy saját API kulcsot.
* Nyisd meg a letöltött Python fájlt, és cseréld ki az api_key = "IDE_JON_AZ_API_KULCS" sort a saját kulcsodra.

3. Indítás
Futtasd le a fájlt a kedvenc kódszerkesztődben vagy a terminálból, és keress rá egy városra!

### ⚠️ Fontos Megjegyzés
Biztonsági okokból a publikus tárolóba nem töltöttem fel a saját API kulcsomat. A kód csak akkor működik, ha az utasításoknak megfelelően megadod a sajátodat.

## 🇬🇧 English Version

A simple yet visually appealing graphical (GUI) weather application built with Python and the OpenWeatherMap API. Based on the entered city, the program fetches the current temperature and weather conditions, and even provides a smart clothing recommendation based on the data!

### 🚀 Features
* Modern UI: Clean, dark-themed design powered by the customtkinter library.
* Real-time Data: Live weather updates from anywhere in the world.
* Smart Assistant: Personalized clothing advice depending on the outdoor temperature.

### 🛠️ Installation & Usage

You need to have Python installed on your machine to run this program.

1. Install Dependencies
Open your terminal or command prompt and run the following command to install the required modules:
pip install requests customtkinter

2. Setup API Key
* Sign up for a free account at the OpenWeatherMap website.
* Generate your own API key.
* Open the Python file in this repository and replace the api_key = "IDE_JON_AZ_API_KULCS" line with your actual API key.

3. Run the App
Execute the script in your favorite IDE or via the terminal and start searching for cities!

### ⚠️ Important Note
For security reasons, I have not included my personal API key in this public repository. The code will only work if you provide your own key as described in the setup instructions.
